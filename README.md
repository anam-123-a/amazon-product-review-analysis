# 🛒 Amazon Product Review Analysis

## 📌 Project Overview
This project analyzes Amazon product review data using Python. The objective is to understand product ratings, customer reviews, and identify top-performing products through Exploratory Data Analysis (EDA).

---

## 🎯 Objectives
- Find the total number of products
- Calculate the average product rating
- Analyze rating distribution
- Identify top-rated products
- Identify most-reviewed products
- Perform basic data cleaning

---

## 🛠️ Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

---

## 📂 Dataset
The dataset contains the following columns:
- Product Name
- Category
- Rating
- Rating Count
- Price
- Discount
- Review

---

## 📊 Analysis Performed
- Dataset Overview
- Data Cleaning
- Missing Value Check
- Duplicate Check
- Statistical Summary
- Average Rating Calculation
- Rating Distribution Analysis
- Top Rated Products
- Most Reviewed Products
- Data Visualization using Matplotlib

---

## 📈 Key Insights
- Total Products: **100**
- Average Rating: **3.71**
- Highest Rating: **5.0**
- Most ratings fall between **3.5 and 5.0**
- No missing values found
- No duplicate records found
